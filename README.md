# FutureNest -- Chennai Real Estate Price Prediction System

A complete Machine Learning system to predict current and future house prices in Chennai using CatBoost and Streamlit.

## Features
• Current house price prediction  
• Future location-based price forecasting  
• High accuracy ML models (R² > 89%)  
• Interactive web UI  
• Clean preprocessing & EDA pipeline 

## Tech Stack
Python, Pandas, Scikit-learn, CatBoost, Streamlit

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Model Accuracy
R² Accuracy : 89.68%  
MAE         : 8.29 Lakhs  
RMSE        : 13.05 Lakhs
